    Bienvenue sur Daaray technologie

    Une école virtuelle gratuite composée de groupe de développeurs très passionnés. L'idée de cette chaîne est de vous aider à pouvoir créer des sites web et des applications de bout à bout facilement avec des astuces simples et chirurgicales, mais aussi de maîtriser toutes les bases en informatique sans dépenser de l'argent. Les cours seront expliqués en Wolof.

    • 01Pourquoi avons nous conçu ces cours ?

      Cours de ce programme, nous allons découvrir des sujets tels que la programmation en HTML, CSS, Js, le développement front-end. Nous avons conçu ces cours pour vous aider à acquérir les compétences dont vous avez besoin pour réussir dans l'industrie du développement web.

    • 02Quelle est l'utilité du tutorielle en ligne ?

      Le tutoriel en ligne est un canal pour apprendre une conception de site Web réactive complète, une animation CSS créative et des effets de Hover Excellent projet Vanilla JavaScrip, conception à l'aide de HTML et CSS Muth plus HTML, CSS3, JavaScrip, jQuery, Bootstrap, Photoshop, conception de site Web

    • 03Quel sont les resultats de ces cours ?

      Nous sommes convaincus que ces cours seront une expérience enrichissante et vous donnera les connaissances et la confiance nécessaires pour devenir un développeur web compétent. Vous aurez également l'opportunité de travailler sur des projets pratiques pour appliquer les connaissances acquises.
